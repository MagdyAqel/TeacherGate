# بوابة المعلم

بوابة عربية لعرض وإدارة التطبيقات التعليمية، مرتبطة بـ Firebase Realtime Database.

- دخول الإدارة الافتراضي: `tech`
- دخول المعلمين الافتراضي: `STD`
- التطبيق متجاوب مع الحاسوب والهاتف.
- يمكن تغيير كلمتي المرور من إعدادات الإدارة.

توجد ملفات التطبيق الكاملة داخل `teacher-portal.zip`، وتقوم صفحة `index.html` بفتحها مباشرة عند النشر عبر GitHub Pages.
