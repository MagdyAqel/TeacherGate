# بوابة المعلم

تطبيق ويب عربي لإدارة روابط التطبيقات التعليمية وعرضها للمعلمين.

## الدخول

- الإدارة: `tech`
- المعلمون: `STD`

يمكن تغيير كلمتي المرور من داخل حساب الإدارة.

## النشر

ملفات التطبيق منشورة مباشرة في جذر المستودع لتعمل مع GitHub Pages بدون أي ملف ZIP.

الرابط المتوقع بعد تفعيل Pages:

https://magdyaqel.github.io/TeacherGate/
